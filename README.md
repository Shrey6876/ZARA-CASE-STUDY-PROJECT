# ZARA STRATEGIC ANALYSIS (Excel Project)

A consulting‑grade, data‑driven analysis of Inditex/Zara’s business model, financial performance, and global expansion, built entirely in Excel using primary Inditex annual reports.

- Rebuilds 20+ years of Inditex financials and KPIs in a single Excel model
- Links quantitative trends to strategy, operations, and international expansion moves
- Designed as a case-study asset for MBA/consulting interviews and applications

**Tools:** Excel, DCF modelling, Inditex annual reports (no Python or code).
